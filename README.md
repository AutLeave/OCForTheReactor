Use OpenComputer to manage the Reactor in GTNH  
通过OC管理GTNH中的核反应堆

## <span style="color:blue">**特点(Character)**</span>
- [<span style="color:red">√</span>] 反应堆检查

## <span style="color:blue">**安装方法(How to use)**</span>
- 下载Start.lua文件
- 使用文档编辑使其配置文件符合你的使用情况
- 导入到游戏OC中的硬盘
- 在游戏OC中启动程序按照指示进行
